Tags: O que são?

(voltar)

As tags são elementos utilizadas para estruturar um documento HTML e interpretadas pelo navegador de acordo com a funcionalidade da tag. Na qual podemos adicionar imagens, links, formatar textos, entre outros. São formadas por colchetes angulares, todas as tags que são abertas precisam ser fechadas. Como por exemplo, <html> </html>. Porém existem algumas tags que não são necessárias fechar, por exemplo, a tag <input/> e a tag <img/>. Mas podemos perceber que essas tags possuem uma barra (/) na própria tag que se inicia. Ou seja, elas mesmas se fecham sozinha, sem a necessidade de uma outra que feche a tag inicial.

O que são os atributos das Tags?

(voltar)

Os atributos das tags fornecem informações adicionais sobre os elementos, são especificados dentro das tags de abertura e interpretados pelo navegador. Na aula foi citado alguns exemplos de atributos globais, como:

  • id: Usado como identificador;
  • style: Contém informações de estilo, utilizado no CSS;
  • class: Usado para identificar um grupo de elementos, também utilizado no CSS;
  • title: Podemos digitar uma mensagem e quando o usúario passa o mouse aparece a mensagem, visualmente traz uma experiência muito legal ao usuário;
  • target: É uma tag âncora, ou seja, define o destino de um link. Os valores mais utilizados dentro dessa tag é o _blank (abre em outra aba) e o _self(abre na mesma aba).
Também existem alguns atributos específicos, como os atributos utilizados na tag img: src (contém um link para a imagem) e o width (utilizado no CSS para definir o tamanho da imagem).

Outras Tags aprendidas na aula

(voltar)

Na aula aprendemos outras tags, como:

  • <head>: Carrega antes de exibir para o usuário;
  • <body>: Onde fica todo o conteúdo que é mostrado para o usuário;
  • <h1>, <h2>, <h3>, <h4>, <h5>, <h6>: Normalmente utilizados para título(h1) e subtítulos (h2). Cada uma dessas tags tem um tamanho de fonte, o h1 é maior(utilizado para título) e vai diminuindo o tamanho até o h6 (o menor);
  • <p>: Representa um parágrafo, utilizado para texto;
  • <i>: Deixa o texto itálico, por exemplo: Olá mundo!!!;
  • <u>: Sublinha um texto, por exemplo: Olá mundo!!!;
  • <strong>: O texto fica em negrito, por exemplo: Olá mundo!!!
  • <mark>: Marcador de texto, por exemplo: Olá mundo!!!
  • <blockquote>: Usado para citação.
